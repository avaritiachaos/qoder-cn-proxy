# Security

This project is intended for local development and learning only.

Do not report or include personal access tokens, cookies, Authorization headers,
local client auth files, or private logs in issues, discussions, pull requests,
or screenshots.

The proxy is designed to listen on `127.0.0.1` only. Do not expose it to public
networks unless you have reviewed and changed the security model yourself.

If you discover a bug that may leak secrets, open a minimal issue that describes
the affected version and reproduction shape without including real credentials.
